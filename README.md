# ✅ 1. What is a Data Structure?

A **Data Structure** is a way to **store and organize data** so it can be used efficiently.

👉 Example: Like arranging books on a shelf so you can find them easily.

---

# ✅ 2. Types of Data Structures

## 🔹 A. Primitive Data Structures

Basic data types:

* Integer (10, 20)
* Float (3.14)
* Character (‘A’)
* Boolean (True/False)

👉 Simple values, no structure inside.

---

## 🔹 B. Non-Primitive Data Structures

More complex structures that store multiple values.

Types:

* Linear
* Non-Linear

---

# ✅ 3. Classification

## 🔸 A. Linear Data Structures

Data is arranged in a **sequence (one after another)**.

Examples:

* Array
* Linked List
* Stack
* Queue

📊 Diagram:

```
[10] → [20] → [30] → [40]
```

👉 Like people standing in a line.

---

## 🔸 B. Non-Linear Data Structures

Data is arranged in a **hierarchical or network form**.

Examples:

* Tree
* Graph

📊 Diagram:

```
      A
     / \
    B   C
```

👉 Like a family tree or social network.

---

# ✅ 4. Explanation of Each Type

### 🔹 Array

* Stores elements in **continuous memory**
* Fixed size

📊 Diagram:

```
Index:  0   1   2   3
Value: [5] [10] [15] [20]
```

👉 Real-life: Seats in a row

✔ Example:
If array = [5,10,15]
👉 array[1] = 10 (access by index)

---

### 🔹 Linked List

* Elements are connected using **links (pointers)**
* Not stored continuously

📊 Diagram:

```
[5|•] → [10|•] → [15|•] → NULL
```

👉 Real-life: Train compartments

✔ Example:
Each node has:

* Data
* Pointer to next node

---

### 🔹 Stack (LIFO: Last In First Out)

* Last inserted item comes out first

📊 Diagram:

```
   [30] ← Top
   [20]
   [10]
```

👉 Real-life: Stack of plates

✔ Example:
Push 10, 20, 30
Pop → removes 30

---

### 🔹 Queue (FIFO: First In First Out)

* First inserted item comes out first

📊 Diagram:

```
Front → [10] [20] [30] ← Rear
```

👉 Real-life: Line at a ticket counter

✔ Example:
Enqueue 10,20,30
Dequeue → removes 10

---

### 🔹 Tree

* Hierarchical structure (parent-child)

📊 Diagram:

```
       10
      /  \
     5    20
```

👉 Real-life: Family tree

✔ Example:

* Root = 10
* Children = 5, 20

---

### 🔹 Graph

* Collection of nodes connected by edges

📊 Diagram:

```
A —— B
|    |
C —— D
```

👉 Real-life: Social network (friends)

✔ Example:

* Nodes: A, B, C
* Edges: A-B, B-C

---

# ✅ 5. Real-Life Summary

| Data Structure | Real-Life Example |
| -------------- | ----------------- |
| Array          | Seats in a row    |
| Linked List    | Train coaches     |
| Stack          | Plates stack      |
| Queue          | Waiting line      |
| Tree           | Family hierarchy  |
| Graph          | Social network    |

---

# 🎯 Final Idea (Easy Way to Remember)

* **Primitive → Single value**
* **Linear → Straight line**
* **Non-linear → Network or hierarchy**
